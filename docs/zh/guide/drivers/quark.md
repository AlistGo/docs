# 夸克网盘

:::tip
由于夸克网盘的限制，下载必须携带 Cookies，所以只能使用本程序中的代理功能进行传输。 （可以用其他机器转）
:::

## **Cookie 和 根文件夹ID**

从请求中获取 Cookie 和根文件夹 ID：
方法：按F12打开“调试”，选中“网络”，找到一个以“sort”开头的文件即可找到
![quark](/img/drivers/quark.png)

或者根目录id也可以从地址栏获取（根目录为0）：

![url](/img/drivers/quark-rootid.png)

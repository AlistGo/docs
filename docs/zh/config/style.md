# 样式设置

### 主色

主要颜色用于页面上。 默认值为`#1890ff`。

### 主页图标

主页图标显示在导航栏的左侧。 默认是`🏠`。

### 主容器

- 最大 980px：主宽度将设置为 `min(99%, 980px)`
- 希望容器：设置内容的最大宽度以匹配当前断点的最小宽度，参考 https://hope-ui.com/docs/layout/container。
